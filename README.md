# كنز الألغاز — بناء APK تجريبي

هذا مستودع عام مخصص فقط لبناء ملف APK تجريبي عبر GitHub Actions المجاني
(بلا حدود على المستودعات العامة)، لأن `almhbob/2026.-` (المستودع الأصلي
الخاص) لا يستطيع تشغيل أي عملية بناء حاليًا بسبب إعدادات الفوترة.

- اللعبة نفسها: `kanz-al-alghaz/index.html`
- غلاف تطبيق أندرويد (Capacitor): `mobile/`
- سير عمل البناء التلقائي: `.github/workflows/android-apk.yml`

كل دفعة (push) على فرع `main` تُشغّل بناء APK تلقائيًا، والناتج يظهر
كـ Artifact باسم `kanz-al-alghaz-debug-apk` في تبويب **Actions**.
